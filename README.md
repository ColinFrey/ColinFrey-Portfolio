Site Description:

Project Concept
Colin Frey Portfolio is a responsive digital art gallery built to showcase original artwork in a clean and modern format. The site focuses on immersive viewing, smooth interaction, and simple navigation.

Key Features and Interaction
Interactive Gallery: Displays artwork in a responsive masonry-style layout.

Search System: Users can search artwork by title or medium with live filtering.

Modal Viewer: Clicking an artwork opens a fullscreen preview with artwork details.

Zoom Functionality: Desktop users can zoom into artwork for closer viewing.

Hover Effects: Artwork tiles reveal titles and mediums on hover.

Motion and Design
Responsive Layout: The gallery adjusts from desktop columns to a single-column mobile layout.

Smooth Animations: Hover effects, modal transitions, and search interactions create fluid navigation.

Minimalist Design: Uses bold typography, dark contrast sections, and clean spacing for a professional gallery feel.

Technical Execution and Accessibility
Keyboard Support: The Escape key closes modals and the search menu.

ARIA Labels: Interactive buttons include accessibility labels for screen readers.

Semantic HTML: Uses organized sections, navigation landmarks, and descriptive alt text.

Assets Used
Fonts: Notable and Shrikhand via Google Fonts.

Libraries: normalize.css and Font Awesome.

Graphics: Original artwork and portfolio images created by the artist.
